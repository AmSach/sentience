# Sentience v3.0 Development Plan

## Architecture
- GUI: PySide6 + WebView for embedded browser
- Backend: FastAPI + SQLite
- LSP: Python-lsp-server + custom adapters
- Hosting: Local FastAPI + ngrok for external
- Skills: Skill registry + 70+ built-in
- Automations: APScheduler + RRule

## Parallel Development Teams
1. Team GUI - IDE workspace, panels, themes
2. Team LSP - Code completion, diagnostics
3. Team Skills - 70+ skills development
4. Team Integrations - OAuth, Gmail, Notion, etc
5. Team Hosting - Local server, domains
6. Team Automations - Scheduler, triggers
7. Team Browser - Playwright automation
8. Team Forms - PDF/DOCX filling
9. Team Context - Project analysis
10. Team Memory - Learning system
